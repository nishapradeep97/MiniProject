# MiniProject
Mini Project in PowerBi
Movie Metadata

1.	Project Overview
    This project focuses on analyzing a movie metadata dataset using Microsoft Excel and Power BI. The goal is to identify trends in movie performance, audience engagement, and key factors influencing IMDB ratings and revenue. The insights derived help in understanding patterns across genres, directors, and actors.

2.	Tools Used
    -	Microsoft Excel
    
    -	Power BI

3.	Dataset
    -	Source: Movie Metadata Dataset 
    
    -	Data contains: 
    
    o	Movie Title
    o	Director Name
    o	Genres
    o	IMDB Score
    o	Gross Revenue
    o	Budget
    o	Duration
    o	Number of Reviews (Critic & User)
    o	Facebook Likes (Actors & Movie)
    o	Language & Country
    o	Release Year

4.	Steps Followed
• Cleaned the dataset in Excel: 
    •	Removed null values and duplicates 
    •	Standardized column formats 
    •	Handled missing data 
    •	Identified missed Director names for the Movie titles
• Imported data into Power BI: 
    •	Created calculated columns and measures such as Total movies, Total revenue, Total budget, Profit, AVG_IMDB
• Built interactive dashboards: 
    •	Bar charts, Donut charts, Treemaps , Line charts, Column Chart, Map
    •	KPIs (Total Movies, Avg IMDB Score, Total Revenue) 
    •	Slicers (Genre, Year, Country)

5.	Key Insights
    •	Movies with higher budgets generally achieved higher gross revenue. 
    •	Certain genres like Action and Adventure tend to have higher IMDB scores. 
    •	Directors with higher Facebook likes often produced more popular movies. 
    •	Most movies are produced in the USA, dominating overall revenue. 
    •	The number of movies released increased significantly after 2000
